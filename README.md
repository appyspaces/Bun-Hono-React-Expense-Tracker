# Expense Tracker Application 
## <forked> Modern React, Bun & Hono Tutorial
[Article][1] [Video][2]

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.25. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
